# Coffee-Shop-Sales-Analysis-
Coffee Shop Sales Analysis -☕📊This project analyzes coffee shop sales data to uncover key business insights such as revenue trends, best-selling products, peak hours, and customer behavior. The dataset is cleaned, explored, and visualized to support data-driven decision-making.  
The dataset contains transactional data including:
- Transaction Date & Time
- Product Name & Category
- Quantity Sold
- Sales Amount
- Store Location
  
🔍 Key Objectives:
- Analyze overall sales performance
- Identify top-selling products
- Understand peak sales hours and days
- Explore customer purchasing patterns
- Detect trends and seasonality in sales
  
This analysis demonstrates how data-driven insights can help improve 
operational efficiency, increase revenue, and enhance customer satisfaction 
in a coffee shop business.
